# Quick Start

Follow these simple steps to quickly get started with DynaBench and start evaluating your machine learning models on the benchmark dataset.

## Prerequisites

Before getting started, ensure that you have the following prerequisites installed on your system:

    Python 3 (>= 3.7)
    pip package manager

## Installation
Install the dynabench package from pip:

    pip install dynabench
